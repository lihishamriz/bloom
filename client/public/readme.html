Store name
Bloom
What are we selling?
We sell flower bouquets from all kinds and colors of flowers.
What additional pages we add?
Gallery - Our gallery of flowers, where every user can add an image.
Club - We let the user the option to join our club and get discounts.
Reviews - We present our reviews, and every user can add a review.
Contact us - User can send us a contact us request using email, and he will get a confirmation email
All the pages can be accessed through URL and through the navbar.
What was hard to do?
It was hard for me to implement the authetication using cookies, because I've never done it before.
It was also challenging to work with react which I wasn't familiar with before.
